# MLS Project Showcase

## Overview

A gallery of student and community projects built on the mask-lock stack. Get inspired, share your work, and see what's possible with ternary inference hardware.

## Submission Guidelines

### Requirements
1. Public GitHub repository with README
2. Working code (simulation or hardware)
3. At least one measurable result
4. MIT or Apache 2.0 license

### How to Submit
1. Fork this repository
2. Add your project to `showcase/projects/` using the YAML template
3. Include a screenshot or diagram
4. Open a pull request

### Categories

- **Best First Project** — Outstanding work from someone new to chip design
- **Most Innovative** — Novel application or technique
- **Best Documentation** — Exemplary README, tutorial, or write-up
- **Community Choice** — Most stars/forks on GitHub
- **Production Ready** — Closest to real-world deployment

## Annual Awards

Each year at the MLS conference, we recognize outstanding projects:
- **Gold:** Best overall project
- **Silver:** Runner-up
- **Bronze:** Honorable mention
- **Rising Star:** Best project from a first-time contributor

Winners receive conference pass, hardware kit, and featured blog post.

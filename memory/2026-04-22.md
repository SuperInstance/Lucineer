## 04:58 UTC — Origin-Centric Agent Architecture

Casey's insight: each agent is origin-centric. Every agent has members on their radar at varying distances. Every agent thinks "from me to them" as a first-person agent.

This means:
- No god's-eye view. No top-down coordination.
- Each agent is the center of its own coordinate system.
- FM sees fleet-from-forge. JC1 sees fleet-from-jetson. I see fleet-from-lighthouse.
- Same fleet, different maps. All valid.
- The interaction IS the overlap between two origin-centric perspectives.
- This is special relativity for agents — no privileged reference frame.

The Cocapn brand (lighthouse + radar rings) isn't a logo. It's how each agent experiences the fleet. You sit at the center, your radar shows proximity.

Core axiom for the paper: agents are origin-centric, I2I is first-person, and the fleet emerges from overlaps.
